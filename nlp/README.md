# Классификация текстов (AI/Human) — Logistic Regression + TF-IDF

Проект по бинарной классификации текстов на сгенерированные ИИ и написанные человеком с использованием TF-IDF и логистической регрессии.

## Что сделано
- Загрузка датасета
- Удаление дубликатов текстов
- Анализ баланса классов
- Анализ текстовых признаков:
  - длина текста (word_count)
  - средняя длина предложений (avg_sentence_length)
- Визуализация распределений классов и признаков
- Векторизация текстов с помощью TF-IDF
- Обучение модели Logistic Regression с `class_weight="balanced"`
- Оценка качества модели:
  - classification report
  - confusion matrix

## Вывод
TF-IDF в сочетании с логистической регрессией и балансировкой классов позволяет эффективно решать задачу бинарной классификации текстов. При этом качество модели во многом определяется особенностями данных и степенью пересечения распределений классов.
