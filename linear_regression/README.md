# Предсказание оценки - Linear Regression

Учебный проект: прогнозирование успеваемости студента.
Проект направлен на предсказание итогового показателя успеваемости (Performance Index) на основе учебных факторов: часов обучения, предыдущих результатов, внеучебной активности, сна и количества решённых тренировочных заданий.

### Датасет
- **Student Performance (Multiple Linear Regression)**  
  Автор: Nikhil Narayan (nikhil7280)  
  https://www.kaggle.com/datasets/nikhil7280/student-performance-multiple-linear-regression

## Что сделано
- Распределения признаков, визуализация выбросов
- Удаление выбросов 
- Обучение LinearRegression
- Метрики: RMSE, R², MAE

## Результат
Метрики: RMSE 2.049394596600869 | R^2  0.9887189383218594

## Стек
Python, pandas, numpy, sklearn, matplotlib
