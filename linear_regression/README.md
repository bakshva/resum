# Предсказание дохода — Linear Regression

Учебный проект: предсказание дохода по возрасту и опыту работы.

## Что сделано
- Распределения признаков, визуализация выбросов
- Удаление выбросов 
- Обучение LinearRegression, сравнение качества с выбросами и без
- Метрики: RMSE, R², MAE

## Результат
| | RMSE | R² |
|---|---|---|
| С выбросами | 1115.5073496751065 | 0.9508709671946177 |
| Без выбросов | 1115.5073496751065 | 0.9508709671946177 |
